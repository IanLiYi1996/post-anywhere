## 发布微信视频号视频 workflow

视频号（channels.weixin.qq.com）使用 wujie 微前端架构，内容在 `<wujie-app>` shadow DOM 内，标准 agent-browser 无法直接访问表单元素。需要 Playwright CDP 配合 agent-browser 完成。

### 前置条件

- Playwright 浏览器已安装：`npx playwright install chromium`
- 系统依赖已安装（Amazon Linux）：`sudo yum install -y atk at-spi2-atk cups-libs libdrm libXcomposite libXdamage libXrandr mesa-libgbm pango alsa-lib nss libxkbcommon`
- `playwright` npm 包：`npm install playwright`
- 视频文件（MP4 H.264，≤20GB，推荐 720p+）

### Step 1: 启动 Chrome 远程调试

必须使用 `--headless=new` 模式（旧 headless 不渲染二维码）：

```bash
CHROME_PATH=$(find ~/.cache/ms-playwright -name "chrome" -path "*/chrome-linux64/*" -type f 2>/dev/null | head -1)

$CHROME_PATH \
  --remote-debugging-port=9222 \
  --no-sandbox \
  --disable-gpu \
  --headless=new \
  --window-size=1920,1080 \
  "https://channels.weixin.qq.com/login.html" &

sleep 5
curl -s http://localhost:9222/json/version
```

### Step 2: 获取二维码让用户扫码登录

```bash
agent-browser --cdp 9222 wait --load networkidle
agent-browser --cdp 9222 wait 8000
agent-browser --cdp 9222 screenshot /path/to/qr-code.png
```

展示截图给用户，等待用户确认扫码登录完成后再继续。二维码约 5 分钟过期。

### Step 3: 验证登录并进入发布页

```bash
agent-browser --cdp 9222 wait --load networkidle
agent-browser --cdp 9222 screenshot /path/to/logged-in.png
```

确认截图显示仪表盘后：

```bash
agent-browser --cdp 9222 snapshot -i
# 找到 button "发表视频" [ref=eN]
agent-browser --cdp 9222 click @eN
agent-browser --cdp 9222 wait --load networkidle
agent-browser --cdp 9222 wait 3000
```

### Step 4: 上传视频（Playwright filechooser）

上传按钮在 wujie shadow DOM 内，必须用 Playwright filechooser 事件拦截：

```javascript
// upload-video.js
const { chromium } = require('playwright');

(async () => {
  const browser = await chromium.connectOverCDP('http://localhost:9222');
  const page = browser.contexts()[0].pages()[0];

  const [fileChooser] = await Promise.all([
    page.waitForEvent('filechooser', { timeout: 10000 }),
    page.evaluate(() => {
      const shadowEl = document.querySelector('wujie-app');
      const sr = shadowEl.shadowRoot;
      const fileInput = sr.querySelector('input[type="file"]');
      fileInput.click();
    })
  ]);

  await fileChooser.setFiles('/path/to/video.mp4');
  console.log('Video uploaded!');
  await page.waitForTimeout(10000);
})().catch(e => console.error(e.message));
```

也可使用项目内 `scripts/publish.js` 一次性完成上传+填写。

### Step 5: 填写视频描述（Playwright frame 访问）

描述编辑器在 wujie shadow DOM 边界内的 `contenteditable` div：

```javascript
const mainFrame = page.frames().find(f => f.url().includes('platform/post/create'));
const editor = await mainFrame.$('.input-editor');

if (editor) {
  await editor.evaluate((el, text) => {
    el.focus();
    el.textContent = text;
    el.dispatchEvent(new Event('input', { bubbles: true }));
  }, '视频描述 #话题1 #话题2');
}
```

### Step 6: 填写短标题

短标题在 shadow DOM 外，可用 agent-browser：

```bash
agent-browser --cdp 9222 snapshot -i
# 找到 textbox "概括视频主要内容，字数建议6-16个字符" [ref=eN]
agent-browser --cdp 9222 fill @eN "短标题"
```

**重要**：短标题不超过 16 个中文字符。

### Step 7: 发表

```bash
agent-browser --cdp 9222 snapshot -i
# 找到 button "发表" [ref=eN]
agent-browser --cdp 9222 click @eN
agent-browser --cdp 9222 wait 10000
agent-browser --cdp 9222 screenshot /path/to/result.png
agent-browser --cdp 9222 get url
```

成功后 URL 跳转到 `https://channels.weixin.qq.com/platform/post/list`。

### Step 8: 清理

```bash
pkill -f "chrome.*remote-debugging" 2>/dev/null
```

## 辅助脚本

项目内 `scripts/publish.js` 可一次完成 Step 4-6：

```bash
node scripts/publish.js --video /path/to/video.mp4 \
  --description "描述内容 #话题" \
  --title "短标题" \
  --port 9222 \
  --screenshot-dir ./screenshots
```

## 视频规格

| 项目 | 要求 |
|------|------|
| 格式 | MP4 (H.264) |
| 最大体积 | 20GB |
| 最大时长 | 8 小时 |
| 分辨率 | 720p+ 推荐，竖屏 1080x1920 |
| 宽高比 | 9:16（竖屏）或 16:9（横屏） |
| 帧率 | ≤60fps |

## 常见问题

| 问题 | 解决方案 |
|------|----------|
| 二维码不渲染 | 必须用 `--headless=new`，不能用 `--headless` |
| `libatk-1.0.so.0` 缺失 | 安装系统依赖（见前置条件） |
| `agent-browser upload` 失败 | 视频号需用 Playwright filechooser（Step 4） |
| 描述填不进去 | 需用 Playwright frame 访问（Step 5） |
| "标题超过16字限制" | 短标题必须 ≤16 字符 |
| 发表按钮灰色 | 检查标题是否超长、必填字段是否遗漏 |
